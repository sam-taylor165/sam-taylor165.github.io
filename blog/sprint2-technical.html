What is the difference between

Margin, Border, and Padding?

Margins

In HTML, a margin defines the space around an element on the page. The direction of this space can be set using the following properties.

  • margin-top
  • margin-bottom
  • margin-left
  • margin-right

    • Or to set a margin in all directions, you could simply use margin.

    Margins can be adjusted through multiple units of measurement, these being pixels (px) or percentage (%). One other value this property can be set by is auto. This allows the browser to determine the size of the margin based on the browsers size.

    Borders

    Another HTML element used when defining space is border. This is used to mark space around an element on the page through the use of three main properties.

  • border-color
  • border-style
  • border-width

    • Border color allows you to change the color of your border, using a hexadecimal color code.


    Border style allows you to change the look of your border. Some styles this property includes are solid, dashed line, double line or a dotted line.


    Border width allows you to set the width of an elementâ€™s borders. The value of this property can be set in pixels (px), point (pt) or cm. Otherwise it can be set to either thin, medium or thick.

    Padding

    The padding element allows you to set the amount of space between the content of an element and its border.

    The value of this property can be defined using either a percentage, length or the word inherit. If the value is set to inherit then the padding will have the same value as its parent element. If a percentage is used it will be a percentage of the containing box. The following properties can be used to define each side of the boxes padding.

  • padding-top
  • padding-bottom
  • padding-left
  • padding-right

    • Otherwise padding can be used to define the space within an element in every direction.
